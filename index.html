MATE
This project named MATE is
    An Utility application
which is very useful for "Individuals" who are doing tasks like Using AI Assistant, creating Simple* Word Files frequently, Often works with Programming [ HTML, CSS, JS ], Often plan to go somewhere out.

This is not a Single project but it has more than one concepts that are covered. Because, there might be several kind of people may use this application and their needs may vary. As a solution for this, I have added some commonly used features among them.

FRONTEND : React Js with Vite Library
BACKEND : Node Js
DATABASE MongoDB Cluster


This application has an Interactive Frontend works. Thanks to React Js with its Vite Library which enabled our application development fast and efficient. We often used Tailwind CSS to make the CSS works more easy and eye-catchy. Most of our works are based on the frontend; it needs a lot of work in frontend rather than in backend.



This application has a significantly smaller backend works such as the Node JS scripts. It is used whenever the user access the Login Page or Signup Page. The backend takes care of the things about the authentication processes such as Sending OTP to their EMAIL and verifies them, Password Checking and Routing.







This application was powered with MongoDB Database management which is helpful in User Signup or Login of existing users.
We had used MongoDB as the Database for this app but later, we have the idea to make Software Application and Mobile Application versions of us.
So, we felt that MongoDB will not be a better choice then, I made a Cluster account for MongoDB and the Database is in the Cloud so the users can access their login details from any device , anywhere.







This project needs Node JS, React JS, Vite Library to be installed to run.
Don't worry, I've embedded them in our package.json file so that you simply open and use the app without installing mannually.
I'm working in ACER ASPIRE 3 which has 16GB RAM and 512GB ROM powered by AMD Ryzen 5 7430U with Radeon Graphics.







We included all the Features as individual Components because it is very easy to reuse them in other codes if needed and they are independent so we just create them and manage them easily. The features ( which are currently available ) are listed below :



1. AI Assistant,
2. Live Code Editor [Html, Css, Js],
3. Text Editor,
4. AI Image Generator,
5. Maps ( Alone & Integrated with AI ),
6. Mini Music Player,
