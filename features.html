AI Assistant

Build: It is built with our own UI Design and powered by Official Gemini AI pastly named Bard AI tool Integration. Thanks to APIs which helps us in integrating the AI Model to my project.

Advantages: Using this AI assistant, you can skip the login process and ads to directly access responses and information.

Disadvantages: The free API used for cost-cutting lacks performance in AI imagine and image generation features.

Live Code Editor

Build: The frontend is designed using React JS and CodeMirror library for syntax highlighting and live preview.

Advantages: No need for extensions like Live Server; offers real-time previews of code.

Disadvantages: Users need to download code files to save them as it uses a virtual engine for preview.

Text Editor

Build: A simple virtual word processor for tasks like creating documents, aligning text, and formatting.

Advantages: Easy to use for quick tasks without needing full-fledged software like MS Word.

Disadvantages: Users need to download files for local storage.

AI Image Generator

Build: Utilizes Hugging Face's pre-trained Stable Diffusion model for image generation.

Advantages: Generate images for educational or personal purposes without external tools.

Disadvantages: Limited performance due to free-tier usage of Hugging Face's servers.

Maps

Build: Uses Nominatim API for location search and Leaflet.js for map rendering.

Advantages: AI integration allows for easy navigation suggestions.

Disadvantages: Limited free map data availability.

Mini Music Player

Build: Lightweight music player for local storage files with a custom UI.

Advantages: Simple and effective for entertainment during work or leisure.

Disadvantages: Cannot stream online music currently.
